# Vant React

基于 [Vant][vant] 和 React 的移动 UI 组件。

## 特性

- 覆盖移动端各类场景
- 开箱即用的高质量 React 组件。
- 完善的文档和示例
- 支持按需引入
- 支持主题定制
- 支持 TypeScript
- 共享 [vant][vant] 设计工具体系。

## 安装

```bash
$ npm install --save @zhengxs/rc-vant
# or
$ yarn add @zhengxs/rc-vant
```

## 浏览器支持

- `Chrome >= 51`
- `iOS >= 10`
- `Android 4.0+`
